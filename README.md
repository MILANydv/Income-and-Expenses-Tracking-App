# To start project
1.  Start Virtualenv env
2.  python manage.py makemigrations
3.  python manage.py migrate
4.  python manage.py runserver
# To run Tests
1.  start project
2.  python manage.py test
# Super user creation
1.  python manage.py createsuperuser
